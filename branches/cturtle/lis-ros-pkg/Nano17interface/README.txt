Interface to ATI Nano17 force/torque fingertip sensors (as mounted on the Barrett Hand).  Uses Barrett's btserial serial library.

In this directory:

lib: the C library files containing the interface

standalone_test: a non-ROS sample test script 

ROS: ROS server and sample cpp and python test clients

bin: where the ROS server and cpp test client are built to

include: .h files for the library

srv: ROS service
